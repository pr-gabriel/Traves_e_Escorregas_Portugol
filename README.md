# Futebel_Traves_E_Escorregas 
Projeto acadêmico com o intuito de criar um jogo de tabuleiro com base nos conhecimentos ensinados, e o conhecimento do grupo. O jogo consiste em um tabuleiro de 25 casas, com o tema futebol, entre as casas existem penalidades e méritos que influenciam na jogatina, ganhado ou perdendo casas. O código utiliza: variáveis, tipos de dados, condicionais, laços de repetição, funções e arrays.
